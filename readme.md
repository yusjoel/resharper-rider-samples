# 关于翻译

## ReSharper命令
1. 由于ReSharper并没有提供中文翻译, 所以相关的命令都会使用原文.
仅在第一次介绍的时候提供翻译.

2. 如果Visual Studio有类似的命令, 那么会保持翻译上的统一.

3. 按键使用原文, 如Escape, 不会改成Esc或其他.

## 词汇表
| 英文  | 翻译 | 备注 |
|-------|------|------|
| Go to | 转到 | 和VS翻译保持一致|
| Navigate | 导航 |和VS翻译保持一致|
